Selectors and filters provide a mehtod to find and extract information

Selectors

Selectors are used to select parts of a webpage using common CSS syle syntax

An example would be p< >to select this we would use ("p) and this weill select all paragraph tags. Once selected we can operate and manipulates them.

Filters

Filtesrs are used to refine the results returned from a selector.

An example would be: ("p:first") this will select the first paragraph tag in the returned set.
